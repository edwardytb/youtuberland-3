GeyserMC


Permite que los jugadores de minecraft bedrock
se unan a un servidor de minecraft java, la serie
es compatible con las versiones desde la
1.19 hasta la 1.19.40.






BetterSleeping


Con este plugin solo se requiere la mitad de
jugadores para dormir



BlockLocker


Este plugin permite protejer cofres, puertas,

etc. Mediante carteles. Para protejer un cofre

debes poner un cartel al cofre no debes de estar

agachado y listo si quieres compartir el acesso

al cofre debes colocar lo siguiente:

/blocklocker 3 nombre











Better Teams


para crear un clan es:

/team create ejemplo

para agregar a alguien:

/team invite jugadores

para aceptar:

/team join ejemplo

para chat del clan

/team chat

para volver a hablar en el chat global es:

/team chat

para poner un home:

/team sethome

para ir al home:

/team home
